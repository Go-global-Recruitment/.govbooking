Convert GoGlobal Booking to Desktop App

Turn your web application into a standalone desktop application

Your project: https://goglobal-recruitment.github.io/.govbooking/

Step-by-Step Instructions

Electron
Tauri
PWA
1

Install Node.js and npm

Download and install Node.js from the official website, which includes npm.

Download Node.js
2

Create a new directory for your desktop app

Open your terminal and run:

Terminal Command 
mkdir govbooking-desktop
cd govbooking-desktop
3

Initialize your project

Run the following command to create a package.json file:

Terminal Command 
npm init -y
4

Install Electron

Install Electron as a development dependency:

Terminal Command 
npm install electron --save-dev
5

Create main.js

Create a file named main.js with the following content:

main.js 
const { app, BrowserWindow } = require('electron');
const path = require('path');

function createWindow() {
  const mainWindow = new BrowserWindow({
    width: 1200,
    height: 800,
    webPreferences: {
      nodeIntegration: true,
      contextIsolation: false
    },
    icon: path.join(__dirname, 'icon.png') // optional
  });

  // Load your GoGlobal Recruitment booking website
  mainWindow.loadURL('https://goglobal-recruitment.github.io/.govbooking/');
}

app.whenReady().then(createWindow);

app.on('window-all-closed', () => {
  if (process.platform !== 'darwin') {
    app.quit();
  }
});

app.on('activate', () => {
  if (BrowserWindow.getAllWindows().length === 0) {
    createWindow();
  }
});
6

Update package.json

Add a start script to your package.json:

package.json 
"scripts": {
  "start": "electron .",
  "build": "electron-builder"
}
7

Run your desktop app

Start your application with:

Terminal Command 
npm start
1

Install Rust

Install Rust using rustup:

Terminal Command 
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
2

Install Tauri prerequisites

Follow the platform-specific instructions:

View Prerequisites
3

Create a Tauri app

Create a new Tauri project:

Terminal Command 
npm create tauri-app@latest govbooking-desktop
cd govbooking-desktop
4

Configure Tauri

Update the tauri.conf.json file to load your web app:

tauri.conf.json 
"windows": [
  {
    "title": "GoGlobal Booking",
    "width": 1200,
    "height": 800,
    "resizable": true,
    "fullscreen": false,
    "url": "https://goglobal-recruitment.github.io/.govbooking/"
  }
]
5

Run your Tauri app

Start your application with:

Terminal Command 
npm run tauri dev
1

Create a Manifest File

Create a manifest.json file in your project:

manifest.json 
{
  "name": "GoGlobal Booking",
  "short_name": "GovBooking",
  "description": "GoGlobal Recruitment Booking System",
  "start_url": "https://goglobal-recruitment.github.io/.govbooking/",
  "display": "standalone",
  "background_color": "#1a2a6c",
  "theme_color": "#fdbb2d",
  "icons": [
    {
      "src": "icon-192.png",
      "sizes": "192x192",
      "type": "image/png"
    },
    {
      "src": "icon-512.png",
      "sizes": "512x512",
      "type": "image/png"
    }
  ]
}
2

Add to Home Screen

Users can install your PWA by selecting "Add to Home Screen" in their browser.

3

Service Worker (Optional)

For offline functionality, register a service worker:

sw.js 
self.addEventListener('install', (event) => {
  console.log('Service Worker installing.');
});

self.addEventListener('fetch', (event) => {
  event.respondWith(
    caches.match(event.request).then((response) => {
      return response || fetch(event.request);
    })
  );
});
